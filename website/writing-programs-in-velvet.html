Writing Programs in Velvet

Hello World Example

fn main() {
    print("Hello, Velvet!");
}

This simple program outputs "Hello, Velvet!" to the console, showcasing Velvet's clean syntax.

Variables and Types

Velvet supports a variety of data types with intuitive declarations.

let x: int = 42;
let name: string = "Velvet";
let is_active: bool = true;

Functions

Create reusable functions with ease:

fn add(a: int, b: int) -> int {
    return a + b;
}

fn greet(name: string) -> string {
    return "Hello, " + name + "!";
}
